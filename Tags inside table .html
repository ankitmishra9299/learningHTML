USE OF TAGS INSIDE TABLE IN HTML

Introduction

  1. The tag defines an HTML table. An HTML table consists of the "table" element and one or more "tr","th", and "td" elements. The "tr" element defines a table row, the "th" element defines a table header, and the "td" element defines a table cell. A more complex HTML table may also include "caption", "col", "colgroup", "thead", "thfoot", and "tbody" elements.
